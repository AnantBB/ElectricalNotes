Project development process for Electrical Notes project

             This process starts by creating one folder at suitable location on the drive where python is installed. If we open any folder in such drive in terminal and type in python then python's interactive window starts. Once folder is created open the folder using vs code. Then at the terminal where folder name is shown at terminal prompt type in virtualenv venv and press enter. Folder named venv gets created in the folder. Now open the folder in file explorer and go to Scripts and activate, right click activate file and copy the path. Now since our terminal is at our newly created folder, only path beyond venv ia required to be pested in the terminal. On pesting appropiate path press enter so terminal will show that it is working in the newly created virtual environment.

             Now we shall be creating new project using django-admin startproject ElectricalNotes name of this project is Electrical Notes. After project creation process completes we see new folder gets created along with venv folder inside our root folder from where we have started. Note that our terminal works in virtual environment whose name is venv. Our environment do not have django installed so we will execute the command pip install django so django along with some other packages gets installed in our virtual environment. Now shift to the folder Electrical Notes using the command cd ElectricalNotes and run the command python manage.py startapp Elements so new application gets created. In this way we have two folders venv and ElectricalNotes in our root folder and ElectricalNotes and Elements folders inside ElectricalNotes folder created. Along with this one project and one application is created.

             Now we are expecting to view a page of our application. The process explained now overrides default page view. As shown below the code inside the view.py file is changed.
views.py
